# getBoundingClientRect
Element.getBoundingClientRect()方法返回元素的大小及其相对于视口的位置  

```js
{
  bottom: -179,
  height: 803,
  left: 0,
  right: 1027,
  top: -982,
  width: 1027,
  x: 0,
  y: -982
}
```
上下左右宽高XY

### DOMRect 对象包含了一组用于描述边框的只读属性——left、top、right和bottom，单位为像素。除了 width 和 height 外的属性都是相对于视口的左上角位置而言的。
